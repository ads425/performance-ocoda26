Integration Settings — Admin Only
Webhook Endpoint

Configure in Make.com / Gateway Hub. All monetary values must be in SAR ر.س.

https://api.performanceocoda.sa/webhooks/v1/metrics/inbound
API Token

Include as Authorization: Bearer <token> in every request.

sk_live_••••••••••••••••••••••••••••
Payload Schema

POST with Content-Type: application/json. spend_sar must be in SAR ر.س.

{
  "client_id":  "CL-001",        // Required
  "platform":   "meta",          // meta | google_ads | tiktok | snapchat | linkedin
  "date":       "2025-06-15",    // ISO 8601
  "period":     "monthly",
  "currency":   "SAR",           // Always SAR ر.س
  "metrics": {
    "spend_sar":   18450.00,     // Actual spend ر.س
    "results":     312,          // Leads / conversions / etc.
    "impressions": 845000,       // Optional
    "clicks":      12400         // Optional
  }
}
